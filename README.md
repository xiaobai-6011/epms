# EPMS 企业项目管理系��

基于 Ant Design Pro + Go + Gin 构建的企业级项目管理SaaS系统。

## 系统架构

### 技术栈

| 层级 | 技术选型 | 说明 |
|------|----------|------|
| 前端框架 | React 18 + TypeScript | 现代前端框架 |
| UI组件库 | Ant Design Pro 6.x | 企业级UI组件库 |
| 后端框架 | Go + Gin | 高性能Web框架 |
| 数据库 | MySQL 8.0 | 关系型数据库 |
| ORM | GORM | Go ORM框架 |
| 认证 | JWT | 无状态令牌认证 |

### 项目结构

```
project-management/          # 后端 (Go)
├── cmd/server/            # 入口文件
├── internal/
│   ├── handlers/          # 处理器
│   ├── models/           # 数据模型
│   └── services/         # 业务逻辑
└── uploads/              # 文件上传目录

project-management-react-v2/ # 前端 (React)
├── src/
│   ├── pages/            # 页面组件
│   ├── services/          # API调用
│   └── locales/          # 国际化
└── config/               # 配置文件
```

### 端口配置

- 后端: `8081`
- 前端: `8003`

## 快速开始

### 后端启动

```bash
cd project-management/backend
./server
```

### 前端启动

```bash
cd project-management-react-v2
npm install
npm run start
```

### 访问地址

- 前端: http://localhost:8003
- 后端: http://localhost:8081
